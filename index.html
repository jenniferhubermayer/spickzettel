Cheats für Git/Github und Terminal

git branch= Branch wird angezeigt also wo ich mich befinde

  • git add * = Alles soll hochgelaaden werden in git
  • git commit -m "content" = In git mit Namen der Änderungen hinterlegt
  • git push = Hochladen
  • git pull = Aktuellen Stand
  • git branch = In welchen branch befinde ich mich ?
  • git branch Corinna = Ich wechsel in den branch Corinna
  • git push -u origin Corinna = Branch wird hochgeladen
  • git checkout master = In den branch Master wechseln
  • git merge Corinna = Die Branch Corinna wird zum Master hinzugefügt
