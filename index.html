Cheats für Git/Github und Terminal

Git-Befehle

Wie lade ich etwas hoch?

  1. git add * (Statt des Sternchens können auch gezielt Dateien genannt werden, die hochgeladen werden sollen.
  2. git commit -m "NACHRICHT" (Hier dann verständliche Erklärungen für die anderen Entwickler*innen als Nachricht dalassen.)
  3. git push (Dieser Befehl lädt die Dateien dann hoch.)

Branch

  • git branch NAME (Um eine neue Branch zu erstellen.)
  • git push -u origin NAME (Damit pushed man die neu erstellte Branch in GitHub.)
  • git branch (Um die Branch anzuzeigen, in der man aktuell arbeitet.)
  • git checkout BRANCH NAME (Um den Branch zu wechseln, in dem man Arbeitet.)
  • git pull (Um den aktuellen Stand des des Masters/der Branch in Visual Studio zu laden. Darauf achten, in welcher Branch man sich befindet.)/li>

Merge

  • git merge BRANCH NAME (Um den Arbeitsstand eines Branches in den Master zu implementieren. Dafür muss man sich im Master befinden)
  • git merge origin/master (Um in die eigene Branch die aktuelle Version aus dem Master zu laden. Dafür muss man sich dann auch in der eigenen Branch befinden.)
